# GitHub Issue Seeds

Use these issue titles to set up your 30-day sprint backlog.

## Week 1 Foundation

1. `Day 1: initialize Git and publish repo foundation`
2. `Day 2: clean generated leftovers and tighten repo identity`
3. `Day 3: verify backend setup and improve quick-start`
4. `Day 4: verify frontend setup and document full-stack run flow`
5. `Day 5: add license, contributing guide, and roadmap docs`
6. `Day 6: create milestones, labels, and daily sprint issues`
7. `Day 7: capture initial product screenshots for the README`

## Week 2 Product Polish

8. `Day 8: improve auth UX copy and validation messaging`
9. `Day 9: refine dashboard messaging and feedback states`
10. `Day 10: polish category forms and delete warnings`
11. `Day 11: improve transaction form clarity and descriptions`
12. `Day 12: improve mobile responsiveness for auth and dashboard`
13. `Day 13: improve mobile responsiveness for categories and transactions`
14. `Day 14: publish architecture decisions and implementation notes`

## Week 3 Quality + Deployment

15. `Day 15: expand backend test coverage for registration and seeding`
16. `Day 16: add validation tests for insufficient balance and transfers`
17. `Day 17: complete frontend smoke coverage or manual QA checklist`
18. `Day 18: improve frontend API error handling and surfacing`
19. `Day 19: add reviewer demo flow and sample walkthrough`
20. `Day 20: improve repo visuals and presentation assets`
21. `Day 21: prepare and publish v0.1.0 release notes`

## Week 4 Portfolio Finish

22. `Day 22: prepare backend deployment configuration and docs`
23. `Day 23: prepare frontend deployment configuration and docs`
24. `Day 24: document production environment variables and deploy order`
25. `Day 25: complete security and cleanup pass`
26. `Day 26: add social proof, roadmap polish, and lessons learned`
27. `Day 27: fix the roughest UX or DX issue from testing`
28. `Day 28: run final QA and patch obvious breakages`
29. `Day 29: prepare v0.2.0 or v1.0.0-beta release summary`
30. `Day 30: final portfolio polish and sprint wrap-up`
